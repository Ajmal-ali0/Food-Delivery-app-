**🍔 Food Delivery App (React Native)**

A simple and modern food delivery mobile app built using React Native. The app allows users to browse food items, search, filter by categories, and view popular meals with images, prices, ratings, and delivery time.

**🚀 Features**

- 🏠 Home screen with greeting and location
- 🔍 Search bar for finding food
- 🍕 Food categories (Pizza, Burger, Chicken Wings, Desserts)
- 🍽️ Popular food listing with images
- ⭐ Ratings, price, and delivery time display
- 🎉 Promotional banner section
- 📱 Bottom navigation bar (Home, Search, Cart, Profile)
- 🎯 Category-based filtering

**🛠️ Tech Stack**

- React Native
- JavaScript (ES6)
- React Hooks (`useState`)
- React Native Core Components

**📂 Project Structure**

App.js      → Main application file (UI + logic)

▶️ How to Run

1. Install dependencies:
```bash
   npm install
````
2. Start the app:

   ```bash
   npx expo start
   ```

3. Run on Android/iOS emulator or physical device.
